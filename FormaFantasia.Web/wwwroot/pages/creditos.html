Tecnologias e Bibliotecas
  • Backend
    ASP.NET Core Razor Pages (.NET 10)
    Framework server-side para a aplicação web. Utilizado para todas as páginas Razor e lógica de servidor.
    Obtido em: dotnet.microsoft.com
  • Backend
    ASP.NET Core MVC (API)
    Utilizado para os controllers de API REST (ProdutosController, CategoriasController, EncomendasController).
    Obtido em: dotnet.microsoft.com
  • ORM
    Entity Framework Core 9.x + Pomelo MySQL Provider
    ORM para acesso à base de dados MySQL. Migrations e modelos gerados com EF Core.
    Obtido via NuGet: nuget.org
  • Real-time
    SignalR
    Utilizado para notificações em tempo real entre o servidor e os clientes.
    Incluído no ASP.NET Core — dotnet.microsoft.com
  • Frontend
    HTML5, CSS3, JavaScript (Vanilla)
    Interface estática desenvolvida sem frameworks adicionais. CSS custom properties, CSS Grid, Flexbox e Fetch API.
    Padrões W3C — sem dependências externas
  • Tipografia
    Google Fonts — Cormorant Garamond & Inter
    Cormorant Garamond para títulos e logótipo. Inter para texto corrido e interface.
    Obtido em: fonts.google.com
  • Identidade
    ASP.NET Core Identity
    Sistema de autenticação e autorização. Roles Admin e Cliente, gestão de utilizadores.
    Incluído no ASP.NET Core — dotnet.microsoft.com
